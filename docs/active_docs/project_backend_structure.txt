# Schema Definitions

## Entity: Project (Source: Add_Project Form / PM_Kanban_Projects Report)

This is the main entity representing a solar project.

```javascript

  //   - Processed Name: id

  //   - Raw Example (getRecordById also returns Owner_Name.Phone_Number, Owner_Name.Email which are fetched implicitly)
  //   - Notes: Lookup to the primary contact(s). Processed into multiple fields in Vuex state.

  "Owner_Name.Phone_Number": String, // Availability: both
```
